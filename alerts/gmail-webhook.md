# Gmail Webhook

Stay updated with your scan results in real-time using **Gmail Webhooks**. This integration allows you to receive **instant email alerts** for every completed scan run—delivered directly to your Gmail inbox. Simply configure the webhook by entering your name and Gmail address, and enable the **Scanning run results** option. Once set up, you'll never miss a critical security update, even when you're away from the dashboard.
