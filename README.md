# Quickstart

Welcome to Akto Agentic AI Security! Secure your AI agents, MCP servers, and agentic applications with complete visibility, automated testing, and real-time protection. Get started in minutes by choosing what you need.

***

## Agentic Discovery

Automatically discover and inventory all your AI agents, MCP servers, and tools across your infrastructure.

* [Homegrown Discovery Agents](agentic-discovery/get-started/homegrown-discovery-agents/) - API Gateways, eBPF, Kubernetes, manual import, and MCP Recon
* [Endpoints Discovery Agents](agentic-discovery/get-started/endpoints-discovery-agents/) - Browser extensions, secure web networks, MCP endpoint shield, and agentic shield

***

## Agentic Red Teaming

Test your AI agents and MCP servers with 1000+ specialized security probes. Identify vulnerabilities like prompt injections, tool abuse, and privilege escalation before attackers do.

Import your components to start testing:

* [AI Agents](agentic-red-teaming/get-started/manual-import/ai-agent-import.md) - AWS Bedrock, Azure AI, Databricks, Vertex AI, Watsonx, or custom agents
* [MCP Servers](agentic-red-teaming/get-started/manual-import/mcp-import.md) - Model Context Protocol tools, resources, and prompts
* [AI Models](agentic-red-teaming/get-started/manual-import/ai-model-import.md) - Large language models and custom AI implementations

***

## Agentic Guardrails

Deploy real-time protection for your production agents. Block threats as they occur, enforce schema conformance, and prevent exploits before they impact your systems.

Set up protection proxies:

* [MCP Proxy](guardrails/overview/proxy/akto-mcp-proxy.md) - Real-time protection for MCP servers
* [Agent Proxy](guardrails/overview/proxy/akto-agent-proxy.md) - Real-time protection for AI agents
