# Copy Probe Content

{% @arcade/embed url="https://app.arcade.software/share/li5uqGbwUsNjBzNLrdKK" flowId="li5uqGbwUsNjBzNLrdKK" %}

Quickly copy probe content for sharing or documentation purposes:

1. Open desired probe in Probe Editor
2. Locate "Copy Content" icon
3. Click the icon
4. Confirmation message appears: "Text copied to clipboard successfully!"

Probe content is now ready to paste elsewhere.

Tip: Use this feature to easily share probe configurations or document your security probes.
