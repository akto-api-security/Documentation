# Deactivate Probe

Akto provides you with a feature to deactivate the custom probe. It ensures that users don't encounter any inaccurate or obsolete results, thereby maintaining the integrity of the security probing process.

Additionally, deactivation facilitates maintenance and updates by preventing false results or errors during system updates or modifications.

In the demonstration below, we have deactivated the probe named **”Broken Authentication by removing the auth token.”**

Go to the **Probe Editor.** Select a **Custom probe** from the **Custom Probe Collection** that you want to deactivate. Click on the **”x”** button to inactive the probe.
