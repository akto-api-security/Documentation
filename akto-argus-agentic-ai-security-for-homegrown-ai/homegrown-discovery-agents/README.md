# Homegrown Discovery Agents

Deploy discovery agents directly in your infrastructure to continuously monitor and discover AI agents, MCP servers, and tool integrations across your production and staging environments.

* [API Gateways](api-gateways/) - Connect to NGINX, Kong, Envoy, AWS API Gateway, Azure API Management, and 15+ more gateways
* [eBPF](ebpf/) - Kernel-level traffic monitoring without modifying applications or infrastructure
* [Kubernetes](kubernetes/) - Native discovery for agents running in Kubernetes clusters
* [Manual Import](manual-import.md) - Quickly import specific components for testing without full infrastructure setup
* [MCP Recon](mcp-recon.md) - Specialized discovery for Model Context Protocol servers, tools, and resources
