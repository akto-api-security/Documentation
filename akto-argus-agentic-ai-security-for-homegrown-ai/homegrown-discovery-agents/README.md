# Homegrown Discovery Agents

Deploy discovery agents directly in your infrastructure to continuously monitor and discover AI agents, MCP servers, and tool integrations across your production and staging environments.

* [API Gateways](/broken/pages/5cQGl9ICiAsmPErYVuoj) - Connect to NGINX, Kong, Envoy, AWS API Gateway, Azure API Management, and 15+ more gateways
* [eBPF](../connectors/others/ebpf/) - Kernel-level traffic monitoring without modifying applications or infrastructure
* [Kubernetes](../connectors/others/kubernetes/) - Native discovery for agents running in Kubernetes clusters
* [Manual Import](../connectors/others/manual-import.md) - Quickly import specific components for testing without full infrastructure setup
* [MCP Recon](../connectors/others/mcp-recon.md) - Specialized discovery for Model Context Protocol servers, tools, and resources
