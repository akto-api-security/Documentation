# MCP Recon

MCP Recon helps you **discover and catalog MCP-compatible servers** across your infrastructure by scanning defined IP ranges. It‚Äôs designed to give you visibility into all MCP endpoints available in your network so Akto can bring them into your security workflow.

### ‚öôÔ∏è Setup

1. Open **Akto Dashboard ‚Üí Quick Start ‚Üí MCP Recon**
2. Click **Connect**
3. Enter IP ranges in either format:
   * CIDR: `192.168.1.0/24`
   * Range: `10.0.0.1-10.0.0.255`
   * Multiple ranges: `192.168.1.0/24, 10.0.0.1-10.0.0.255`
4. Click **Discover**
