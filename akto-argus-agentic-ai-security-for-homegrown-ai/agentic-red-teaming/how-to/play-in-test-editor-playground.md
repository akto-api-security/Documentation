# Play in Probe Editor Playground

Play in Probe Editor Playground Akto Probe Editor allows you to create and customize agentic AI security probes. It gives you access to over 100 built-in probes and uses YAML for easy writing and understanding. It also lets you probe against a sample API for real-time feedback. This is perfect for both finding prebuilt security probes and creating custom ones.

In the demonstration below, we use Akto's Probe Editor to run a pre-built probe against the **`rest/captcha`** endpoint of the **`juice_shop_demo`** collection to detect any potential vulnerabilities within the API.
