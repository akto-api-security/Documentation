# Edit Probe Settings

From the probe results page, you can edit probe settings and re-run probes directly without creating a new probe run.

## Editing and Re-running Probes

1. Navigate to **AI Red Teaming > Results**
2. Select the probe run you want to modify
3. Click **Edit probe settings**
4. Adjust parameters such as:
   - Probe categories
   - Rate limits
   - Authentication settings
   - Target components
5. Click **Re-run probe** to execute with updated settings

This allows quick iteration on probe configurations without starting from scratch.
