# View New Parameters

Akto automatically updates Agentic Inventory, whenever a new Agentic component has been added to your agentic collection and also updates the parameters it contains. If there is any sensitive parameter has been contained by any new Agentic component, Akto automatically updates it in Agentic component changes. You can view new parameters and new sensitive parameters like **`Passwords`** or **`Email ID`** that you need to keep safe. So with Akto, you always know what's going on with your Agentic components and can stay safe from unwanted surprises.
