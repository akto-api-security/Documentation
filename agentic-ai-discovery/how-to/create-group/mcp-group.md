# Create Agentic Group

{% @arcade/embed flowId="wveTzZ9bqLIbxHpkkPBJ" url="https://app.arcade.software/share/wveTzZ9bqLIbxHpkkPBJ" %}

You can create Agentic groups in two ways:

1. From existing Agentic components:
   * Select the desired Agentic components from your Agentic Inventory
   * Choose "Add to Agentic group" from the available actions
   * Select "New Agentic group" and provide a name for the group
2. Create a new group:
   * Select the desired Agentic components from your Agentic Inventory
   * Select "Create new group"
   * Provide a name for the group and add Agentic components to the group.
