# Manual Import

Directly import your AI agents, MCP servers, and AI models into Akto for immediate security testing without deploying discovery agents or connecting to your infrastructure.

* [AI Agent Import](../../../akto-argus-agentic-ai-security-for-homegrown-ai/agentic-red-teaming/get-started/manual-import/ai-agent-import.md) - Import agents from AWS Bedrock, Azure AI Foundry, Databricks, Vertex AI, Watsonx, or custom implementations
* [MCP Import](../../../akto-argus-agentic-ai-security-for-homegrown-ai/agentic-red-teaming/get-started/manual-import/mcp-import.md) - Import Model Context Protocol servers by connecting to their SSE endpoints
* [AI Model Import](../../../akto-argus-agentic-ai-security-for-homegrown-ai/agentic-red-teaming/get-started/manual-import/ai-model-import.md) - Import models like Gemini, OpenAI, Claude, DeepSeek, Llama, Grok, or custom implementations
