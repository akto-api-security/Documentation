# Manual Import

Directly import your AI agents, MCP servers, and AI models into Akto for immediate security testing without deploying discovery agents or connecting to your infrastructure.

* [AI Agent Import](../../../getting-started/agentic-red-teaming/ai-agent-import.md) - Import agents from AWS Bedrock, Azure AI Foundry, Databricks, Vertex AI, Watsonx, or custom implementations
* [MCP Import](../../../getting-started/agentic-red-teaming/mcp-import.md) - Import Model Context Protocol servers by connecting to their SSE endpoints
* [AI Model Import](../../../getting-started/agentic-red-teaming/ai-model-import.md) - Import models like Gemini, OpenAI, Claude, DeepSeek, Llama, Grok, or custom implementations

Once imported, [run security tests](../README.md) to identify vulnerabilities, view components in your [Agentic Discovery](../../../agentic-discovery/get-started/) dashboard, or [enable Guardrails](../../../guardrails/overview.md) for real-time protection.
